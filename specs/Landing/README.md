## Directory Convention

One directory per spec, named:

```
Landing-NNN_short-title/
├── BRAINSTORM.md   # Brainstorm
├── PRD.md          # Product Requirements
├── TECH.md         # Technical Design
└── TEST.md         # Test Plan
```
