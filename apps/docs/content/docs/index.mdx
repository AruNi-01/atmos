## What is Fumadocs?

Fumadocs is a documentation generator for Next.js. It is a zero-config solution for generating documentation for your Next.js project. It is built on top of [Fumadocs Core](https://github.com/fumadocs/core) and [Fumadocs MDX](https://github.com/fumadocs/mdx).

## How to use it?

```bash
npm install fumadocs-mdx
```

```bash
npx fumadocs-mdx
```

```bash
npm run build
```

```bash
npm run start
```
