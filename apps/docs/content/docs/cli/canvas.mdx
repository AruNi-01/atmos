`atmos canvas` gives coding agents a way to work with the Canvas board you have open in Atmos.

Most users do not need to run these commands by hand. They are mainly useful when an agent is helping you create notes, arrange cards, draw diagrams, or inspect what is already on the board.

## Before You Use It

1. Open **Canvas** in Atmos.
2. Enable agent access from the Canvas agent controls.
3. Give the agent the Canvas instructions shown in the UI.
4. Keep Canvas open while the agent works.

## What Agents Can Do

- Check whether Canvas is ready.
- Read the current board.
- Create notes, frames, shapes, and arrows.
- Move, align, group, or delete selected items.
- Focus the view on important work.

## Tips

- Use Canvas for planning, comparing workspaces, or explaining architecture.
- Ask the agent to keep changes small and visible.
- If a command fails, make sure Canvas is open and agent access is enabled.
