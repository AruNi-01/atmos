`atmos canvas` 让编码 Agent 可以操作你在 Atmos 中打开的 Canvas 画布。

大多数用户不需要手动运行这些命令。它们主要用于 Agent 帮你创建笔记、整理卡片、绘图，或查看画布上已有内容。

## 使用前

1. 在 Atmos 中打开 **Canvas**。
2. 从 Canvas Agent 控件中允许 Agent 访问。
3. 把 UI 中显示的 Canvas 指令给 Agent。
4. Agent 工作时保持 Canvas 打开。

## Agent 可以做什么

- 检查 Canvas 是否已准备好。
- 读取当前画布。
- 创建笔记、Frame、图形和箭头。
- 移动、对齐、分组或删除选中元素。
- 把视图聚焦到重要工作上。

## 使用建议

- 用 Canvas 做规划、比较工作空间或解释架构。
- 要求 Agent 保持改动小而可见。
- 如果命令失败，确认 Canvas 已打开，并且已允许 Agent 访问。
