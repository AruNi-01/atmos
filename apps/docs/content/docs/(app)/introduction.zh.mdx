# 介绍

Atmos 是一个并发开发工作空间，用来在持久终端会话中运行多个 code-agent。

它会为每个任务提供独立的 Git worktree，让长时间运行的 code-agent 终端保持在线，并让你在多个并行分支之间切换时不丢上下文。你可以把 Atmos 当作桌面应用使用，也可以在浏览器里打开，还可以把界面连接到另一台真正存放代码的机器。

## 你可以做什么

- **同时打开多个任务** — 为每个功能、修复、实验或 Review 创建独立工作空间和分支。
- **并行运行多个 code-agent** — 每个工作空间可以保留一个或多个 Agent 终端。
- **让长时间运行的工作可恢复** — 即使刷新页面、重启应用，甚至电脑宕机重启，终端会话也会保留在对应工作空间上。
- **在上下文所在的地方使用 Agent** — 在工作空间终端里运行你的 code-agent CLI，然后在同一个地方检查文件、diff、Review 和输出。
- **发布前先 Review** — 检查 diff、留下行内评论、让 Agent 修复问题，然后更有把握地提交。
- **可视化规划** — 当多个工作空间需要对比时，把终端和笔记固定到 Canvas。
- **跟踪大量任务** — 用 Kanban、标签、优先级、搜索和 Agent 状态保持方向感。
- **在另一台机器上工作** — 当仓库和 Agent 需要跑在远程机器上时，连接到 Atmos Computer。
