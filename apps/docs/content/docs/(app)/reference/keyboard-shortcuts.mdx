title: Keyboard shortcuts
description: Find the fastest way to move around Atmos.

Keyboard shortcuts help you move through Atmos without leaving the keyboard. The exact keys can vary by platform and version.

## Where to find them

Open **Settings** in Atmos and look for the shortcuts section. It lists the current shortcuts for your version of the app.

## Useful shortcut types

- Open search or the command palette.
- Create or switch workspaces.
- Toggle common panels.
- Move between editor, terminal, and review areas.

## Terminal focus

Some shortcuts also work while the terminal is focused. If a shortcut does not run from the terminal, click outside the terminal once and try again.
