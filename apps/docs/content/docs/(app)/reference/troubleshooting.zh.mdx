---
title: 故障排查
description: 常见问题和修复方式。
---

先从下面这些简单检查开始。它们能解决大多数设置和连接问题。

## Atmos 打不开

1. 重启应用。
2. 如果你通过 CLI 管理 Atmos，运行 `atmos runtime ensure`，确认本地运行环境已启动。
3. 再次打开桌面应用，或在浏览器中打开 [app.atmos.land](https://app.atmos.land)。

## 终端没有响应

1. 如果当前命令卡住了，先停止它。
2. 在同一个工作空间里打开一个新终端。
3. 运行一个小命令，比如 `pwd`，确认 shell 可以正常工作。

## 远程电脑离线

在那台机器上运行 `atmos runtime status`。如果它已经停止，运行 `atmos runtime ensure`。

## Agent 输出看起来不对

确认你打开的是正确的工作空间和项目文件夹。然后用更小、更具体的请求重新运行 Agent。
