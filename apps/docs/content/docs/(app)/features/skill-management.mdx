---
title: Skill Management System
description: Discover, enable, disable, and delete agent skills with one-click control.
---

Skills are reusable instructions that shape how agents behave—review checklists, GitHub flows, doc standards, or team conventions. The skill management system lets you curate which capabilities are active per project or session.

## Why it matters

- Stop retyping the same agent instructions.
- Keep behavior consistent across teammates.
- Turn specialized workflows on only when needed.
- Remove skills that are outdated or too broad.

## What you can do

- Browse available skills in the app.
- Enable or disable skills with one click.
- Delete skills you no longer use.
- Combine skills with [Agent Chat Panel](/features/agent-chat-panel) for focused tasks.

## Get started

1. Open the skills area in Atmos.
2. Review descriptions to understand each skill’s job.
3. Enable skills that match your current project.
4. Disable or remove skills that add noise.

## Tips

- One skill, one job—split oversized skills into smaller ones.
- Revisit skills when you change team process or tooling.
