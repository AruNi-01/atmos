---
title: Agent 状态通知
description: 基于钩子的提醒——完成、失败或需授权时，支持系统通知与自托管推送。
---

Agent 状态通知让你在不必盯着终端时仍知道何时需要介入。Atmos 监听 Agent 钩子，可通过系统通知或自建推送栈送达提醒。

## 为什么重要

- 长时间任务可暂时离开。
- 需要审批时及时收到提醒。
- 任务失败时立刻知晓。
- 通过 ntfy、Gotify 等将提醒发到手机或团队频道。

## 支持渠道

- 桌面或操作系统原生通知。
- 自托管推送：**ntfy**、**Gotify** 或自定义 Webhook。

## 快速开始

1. 打开 Atmos **设置**。
2. 找到通知相关配置。
3. 只启用你会响应的渠道。
4. 运行短任务验证是否收到提醒。

## 提示

- 少而精——噪音会掩盖重要事件。
- 与 [Agent 状态追踪](/zh/features/agent-status-tracking) 配合，应用内状态同步。
