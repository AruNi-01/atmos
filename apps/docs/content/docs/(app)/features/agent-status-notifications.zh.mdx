---
title: Agent 状态通知
description: 当 Agent 完成、失败或需要你处理时收到提醒。
---

Agent 状态通知会在有事情需要你关注时提醒你，这样你不用一直盯着终端。

它适合长时间运行的任务、审批请求和失败提醒。保持通知克制：目标是知道什么时候该回来，而不是把每个小更新都变成提醒。

## 为什么重要

- 长时间的 Agent 任务运行时，你可以离开一会儿。
- 需要审批时及时收到提醒。
- 任务失败时马上知道。
- 配置后可以把提醒发到手机或团队渠道。

## 支持的渠道

- 桌面或系统通知。
- 配置后可使用 **ntfy**、**Gotify** 或自定义 webhook 等推送服务。

## 开始使用

1. 在 Atmos 中打开 **Settings**。
2. 找到通知相关区域。
3. 只启用你真的会响应的渠道。
4. 运行一个短 Agent 任务，确认通知可以送达。

## 使用建议

- 只开启你会处理的提醒，噪音会掩盖重要事件。
- 和 [Agent 状态跟踪](/zh/features/agent-status-tracking) 搭配，查看应用内状态变化。
