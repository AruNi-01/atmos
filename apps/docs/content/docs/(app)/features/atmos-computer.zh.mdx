title: Atmos Computer

**Atmos Computer** 让你的桌面应用或浏览器连接到已注册到 Atmos 的电脑。连接后，项目、终端、Agent 和工作空间都会在所选电脑上运行。

当你希望界面留在笔记本上，而实际工作运行在更强的机器、家里的机器或云主机上时，可以使用它。

## 它做什么

- 将一台电脑注册到你的 Atmos 账户下。
- 当那台电脑的本机 Atmos runtime 在线时，让它保持可用。
- 让当前应用可以在本机工作和已注册电脑之间切换。
- 将工作空间、终端、Canvas、Review 和 Agent 操作路由到所选电脑。

## 典型流程

1. 打开 **Settings** → **Atmos Computer**。
2. 创建或保存 **Access Key**。
3. 注册负责运行工作的电脑。
4. 等它出现在 **My Computers** 中。
5. 点击 **Connect**。

连接后，界面会切换到那台电脑。需要回到眼前这台机器的 runtime 时，使用 **Use locally**。

## This Computer

**This Computer** 区域用于注册你当前正在使用的机器。这样你就可以在另一台设备的 **My Computers** 中看到并连接这台 host。

## My Computers

已注册的电脑会显示在线状态、最后在线时间和连接操作。离线电脑会保留在列表中，但需要重新上线后才能选择。

## 安全注意事项

- 像保护密码一样保护 access key。
- 注册码是短期且一次性的。
- 删除不再信任的电脑。
- Connect session 会过期，需要时重新连接。
