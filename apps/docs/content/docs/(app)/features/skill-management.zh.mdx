---
title: 技能管理系统
description: 一键发现、启用、禁用或删除 Agent 技能，灵活控制能力集合。
---

技能是可复用的 Agent 指令——审查清单、GitHub 流程、文档规范或团队约定。技能管理系统让你按项目或会话 curated 哪些能力处于启用状态。

## 为什么重要

- 避免重复输入相同 Agent 说明。
- 团队成员行为一致。
- 仅在需要时开启专项流程。
- 移除过时或过于宽泛的技能。

## 能做什么

- 在应用中浏览可用技能。
- 一键启用或禁用。
- 删除不再使用的技能。
- 与 [全局 Agent 聊天面板](/zh/features/agent-chat-panel) 组合完成聚焦任务。

## 快速开始

1. 打开 Atmos 中的技能区域。
2. 阅读描述，理解每个技能的职责。
3. 启用与当前项目匹配的技能。
4. 禁用或删除增加噪音的技能。

## 提示

- 一个技能只做一件事——过大的技能应拆分。
- 团队流程或工具变更时重新评估技能列表。
