---
title: 技能管理
description: 选择你希望可用的可复用 Agent 指令。
---

Skills 是给 Agent 的可复用指令。它们可以记录 Review 清单、写作规范、项目习惯或可重复工作流，让你不用每次都重新解释同一件事。

## 为什么重要

- 不用反复输入同样的 Agent 指令。
- 让团队成员之间的行为更一致。
- 只在任务需要时开启专门工作流。
- 移除过时或过于宽泛的技能。

## 你可以做什么

- 在应用中浏览可用技能。
- 一键启用或禁用技能。
- 删除不再使用的技能。
- 和 [Agent 聊天面板](/zh/features/agent-chat-panel) 搭配，处理聚焦任务。

## 开始使用

1. 在 Atmos 中打开 **Skills**。
2. 阅读描述，理解每个技能的用途。
3. 启用符合当前项目的技能。
4. 禁用或移除会增加噪音的技能。

## 使用建议

- 一个技能只做一件事；把过大的技能拆小。
- 当团队流程或工具变化时，重新检查技能。
