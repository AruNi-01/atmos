The global agent chat panel lets you talk to coding agents without living in the terminal. Open it from anywhere in Atmos, describe a task, and let the agent inspect the workspace through **ACP** (Agent Client Protocol), reusing the same Code Agent CLI you already trust.

## Why it matters

- Ask questions while reading files or diffs.
- Hand off focused changes with clear context.
- Avoid copy-pasting between chat apps and the repo.
- Stay in the workspace that owns the task.

## What to ask

- Explain how part of the codebase works.
- Locate where a feature is implemented.
- Make a small change and run the relevant check.
- Review a diff before commit.

## Get started

1. Open the workspace for your task.
2. Open the agent chat panel.
3. State the goal, files, and expected outcome.
4. Review edits and terminal output before you ship.

## Tips

- Give agents a clear finish line; split large work into steps.
- Pair with [Skill Management](/features/skill-management) for repeatable workflows.
