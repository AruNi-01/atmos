title: Global Search & Command Palette
description: Keyboard-driven workflow to find workspaces, features, and commands instantly.
---

Global search and the command palette are your fast lane through Atmos. Jump to a workspace, open Canvas or Kanban, or run a common action without clicking through menus.

## Why it matters

- Keep hands on the keyboard during deep work.
- Open features you use rarely but need quickly.
- Find workspaces by project or task name.
- Reduce friction when managing many parallel tasks.

## Use it for

- Finding a workspace or project.
- Opening Canvas, Kanban, usage, or settings.
- Running built-in commands and shortcuts.
- Switching context during agent-heavy sessions.

## Get started

1. Open the command palette (see [Keyboard shortcuts](/reference/keyboard-shortcuts)).
2. Type a few words from what you want.
3. Select the result and press Enter.

## Tips

- Search with natural words: project name, feature name, or command verb.
- Combine with [Multi-Workspace Development](/features/multi-workspace) when you run many tasks at once.
