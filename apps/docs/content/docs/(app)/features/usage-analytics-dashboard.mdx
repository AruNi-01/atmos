---
title: Usage Analytics Dashboard
description: Track AI subscription quotas, agent token usage, and cost estimates in one view.
---

The usage analytics dashboard shows how much agent work is happening across your workspaces—sessions, tokens, and estimated cost—so you can plan large tasks and catch surprises early.

## Why it matters

- See token consumption before a bill spikes.
- Compare activity across projects and agents.
- Watch subscription or provider limits when available.
- Decide when to switch models or shorten prompts.

## What to check

- Recent agent sessions and duration.
- Token usage by model or provider.
- Estimated cost for the current period.
- Quota headroom for AI coding subscriptions.

## Get started

1. Open the usage or analytics area in Atmos.
2. Review recent sessions after long agent runs.
3. Compare estimates with your provider billing page.

## Tips

- Treat in-app numbers as planning estimates; provider invoices are the source of truth.
- Check usage before kicking off large refactors or batch reviews.
