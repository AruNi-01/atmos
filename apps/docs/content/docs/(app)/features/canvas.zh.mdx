Canvas 是持久、跨项目的画板。可固定任意工作区的终端卡片、添加便签，并让 Code Agent 在 Agent 工作流内调整示意图或布局。

## 为什么重要

- 在同一画面比较多个工作区。
- 架构草图与实时终端输出并排。
- 可视化规划迁移或多仓库协作。
- Agent 可更新便签、示意图与排版。

## 可放置内容

- 不同工作区或项目的终端卡片。
- 目标、风险或决策的便签。
- Agent 生成的示意图与布局。

## 快速开始

1. 从 Atmos 打开 **Canvas**。
2. 固定关心的终端或工作区。
3. 为当前目标写简短便签。
4. 需要时请 Agent 协助整理画板。

## 提示

- 当作工作看板——任务完成后归档卡片。
- 与 [多工作区开发](/zh/features/multi-workspace) 配合并行推进。
