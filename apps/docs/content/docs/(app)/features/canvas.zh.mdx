Canvas 是一个可视化工作板，适合那些跨多个终端、工作空间或想法的工作。你可以固定终端卡片、添加笔记、画流程，也可以在上下文太多不适合列表时，让 Agent 帮你整理画布。

## 为什么重要

- 在同一个界面上对比多个工作空间。
- 把架构草图放在实时终端输出旁边。
- 可视化规划迁移或多仓库工作。
- 让 Agent 更新布局、笔记或图。

## 可以放什么

- 来自不同工作空间或项目的终端卡片。
- 记录目标、风险或决策的便签。
- 由你或 Agent 创建的图和布局。

## 开始使用

1. 从 Atmos 打开 **Canvas**。
2. 固定你关心的终端或工作空间。
3. 为当前目标添加一条简短笔记。
4. 需要时让 Agent 帮你整理或记录画布。

## 使用建议

- 把 Canvas 当作工作板使用；任务完成后归档卡片。
- 和 [多工作空间开发](/zh/features/multi-workspace) 搭配，处理并行工作。
