---
title: 轻量本地模型
description: 用小型本地模型处理轻量辅助任务。
---

轻量本地模型让 Atmos 在你的机器上处理小型辅助任务，而不是把每个请求都发给云端模型。

## 为什么重要

- 生成会话标题时不消耗云端 token。
- 在本地从工作空间笔记中提取任务列表。
- 根据 diff 在设备上起草 Git 提交信息。
- 让轻量推理避开订阅额度。

## 适合什么

- 短文本生成和分类。
- 小输入的摘要。
- 不需要前沿模型的重复性辅助任务。

## 不能替代什么

- 大型重构或多文件推理。
- 需要最新前沿模型的任务；这些任务请使用你的 Code Agent CLI。

## 开始使用

1. 在 Atmos 中打开模型或本地推理设置。
2. 选择一个小模型，并在可用时启动它。
3. 用本地生成处理标题、任务提取或提交信息草稿。
4. 对于重型编码任务，回到云端 Agent。

## 使用建议

- 在笔记本上优先选择量化小模型，速度更快。
- 用电池供电前，先确认 GPU/CPU 要求。
