轻量级本地模型通过 **llama-server** 在本机运行小型 Hugging Face checkpoint——适合不必调用云端 API 的短文本任务。

## 为什么重要

- 生成会话标题而不消耗云端 Token。
- 本地从工作区笔记提取 TODO。
- 根据 diff 在设备上起草 Git 提交说明。
- 轻量推理不占用订阅额度。

## 适合场景

- 短文本生成与分类。
- 小输入摘要。
- 不必用前沿模型的重复性辅助。

## 不适合替代

- 大型多文件重构或复杂推理。
- 需要最新前沿模型的任务——请继续使用 Code Agent CLI。

## 快速开始

1. 打开 Atmos 中的模型或本地推理设置。
2. 选择小模型并启动 **llama-server**（可用时一键启动）。
3. 将本地生成用于标题、TODO 提取或提交说明草稿。
4. 繁重编码工作仍使用云端 Agent。

## 提示

- 笔记本上优先选量化小模型以提速。
- 启用前确认 GPU/CPU 与续航影响。
