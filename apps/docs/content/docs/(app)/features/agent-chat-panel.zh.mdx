全局 Agent 聊天面板让你不必一直待在终端里也能与编码 Agent 协作。在 Atmos 任意位置打开面板、描述任务，Agent 通过 **ACP**（Agent Client Protocol）检查工作区，复用你已信任的 Code Agent CLI。

## 为什么重要

- 阅读文件或 diff 时随时提问。
- 用清晰上下文交接小范围改动。
- 避免在聊天应用与仓库间复制粘贴。
- 始终停留在拥有该任务的工作区。

## 适合提问

- 解释代码库某部分如何工作。
- 定位功能实现位置。
- 做小改动并运行相关检查。
- 提交前审查 diff。

## 快速开始

1. 打开对应任务的工作区。
2. 打开 Agent 聊天面板。
3. 说明目标、涉及文件与预期结果。
4. 发布前审查编辑与终端输出。

## 提示

- 给 Agent 明确的完成标准；大任务分步进行。
- 配合 [技能管理](/zh/features/skill-management) 固化可重复流程。
