---
title: 全局 Agent 聊天面板
description: 在 Atmos 的任何地方和编码 Agent 对话。
---

全局 Agent 聊天面板让你不用离开正在查看的文件、diff、终端或工作空间，也能和编码 Agent 对话。

当你需要一次聚焦的对话时使用它：询问某个文件的作用、请求一个小改动、Review 一个 diff，或把一条笔记变成下一步行动。聊天会绑定到当前工作空间，所以 Agent 能拿到正确的项目上下文。

## 为什么重要

- 一边读文件或 diff，一边提问。
- 带着清晰上下文交给 Agent 做聚焦修改。
- 避免在聊天应用和仓库之间来回复制。
- 始终留在拥有这个任务的工作空间里。

## 可以问什么

- 解释代码库的某一部分如何工作。
- 找到某个功能在哪里实现。
- 做一个小改动，并运行相关检查。
- 提交前 Review 一个 diff。

## 开始使用

1. 打开当前任务的工作空间。
2. 打开 Agent 聊天面板。
3. 说明目标、有用文件和预期结果。
4. 发布前检查改动和终端输出。

## 使用建议

- 给 Agent 一个清晰的终点；大任务拆成小步骤。
- 和 [技能管理](/zh/features/skill-management) 搭配，处理可重复的工作流。
