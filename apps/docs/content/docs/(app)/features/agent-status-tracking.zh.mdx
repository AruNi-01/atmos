Agent 状态追踪让整个 UI 与 Agent 实际行为一致。钩子上报生命周期变化，侧边栏、工作区与徽章保持最新，无需轮询终端。

## 为什么重要

- 看清哪个 Agent 在运行或等待审批。
- 知晓会话何时空闲或结束。
- 多 Agent 协作时不靠猜状态。
- 仅在有意义的状态变化时触发 [通知](/zh/features/agent-status-notifications)。

## 常见状态

- **运行中** — 正在执行工具或命令。
- **空闲** — 已连接，等待输入。
- **等待授权** — 需要你批准才能继续。
- **完成** — 任务结束或已停止。

## 工作原理

Agent 通过 **钩子** 上报状态；Atmos 将近实时同步到工作区界面、Kanban 等相关视图。

## 提示

- 若长期停在「等待授权」，检查终端或聊天中的批准提示。
- 长时间任务离开时，可配合通知使用。
