title: Agent 状态跟踪
description: 查看 Agent 是运行中、空闲、等待中还是已完成。
---

Agent 状态跟踪会在 Atmos 中显示 Agent 正在做什么。当你同时运行多个任务时，状态指示能帮助你找到需要关注的会话。

## 为什么重要

- 看出哪个 Agent 正在运行，哪个卡在审批上。
- 知道某个会话是否已经空闲或完成。
- 管理多个 Agent 时不用猜状态。
- 只在有意义的变化上触发[通知](/zh/features/agent-status-notifications)。

## 你会看到的状态

- **Running** — Agent 正在执行工具或命令。
- **Idle** — 已连接，但正在等待输入。
- **Waiting for permission** — 需要你的批准才能继续。
- **Done** — 任务已完成或停止。

## 在哪里看到

Atmos 会在有助于你做决定的位置显示 Agent 状态：工作空间顶部、底部状态区、Kanban 和相关任务视图。

## 使用建议

- 如果状态一直停在 “waiting”，去终端或聊天里查看审批提示。
- 如果你会在长任务期间离开，建议把状态跟踪和通知搭配使用。
