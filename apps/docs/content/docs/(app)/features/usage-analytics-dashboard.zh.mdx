title: 用量分析看板
description: 跟踪 AI 订阅额度、Agent Token 消耗与费用预估。
---

用量分析看板展示各工作区的 Agent 活动——会话、Token 与预估费用——便于规划大任务并及早发现异常。

## 为什么重要

- 账单飙升前看到 Token 消耗。
- 对比不同项目与 Agent 的活动。
- 在可用时关注订阅或厂商额度。
- 决定何时换模型或缩短提示词。

## 关注指标

- 近期 Agent 会话与时长。
- 按模型或厂商的 Token 用量。
- 当前周期预估费用。
- AI 编码订阅剩余额度。

## 快速开始

1. 打开 Atmos 中的用量或分析区域。
2. 长时间 Agent 任务后查看近期会话。
3. 将预估与厂商账单页面对照。

## 提示

- 应用内数字用于规划与排错；最终以厂商账单为准。
- 大型重构或批量审查前先查看用量。
