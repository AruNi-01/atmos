---
title: 用量分析
description: 在一个视图中跟踪 AI 用量、额度和预估成本。
---

用量分析会显示各个工作空间中发生了多少 AI 工作。用它发现重型会话、比较服务商，并在开始大任务前避免意外。

## 为什么重要

- 在账单激增前看到用量。
- 对比不同项目和 Agent 的活动。
- 在可用时关注订阅或服务商限制。
- 决定何时切换模型或缩短 prompt。

## 要看什么

- 最近的 Agent 会话和耗时。
- 按模型或服务商统计的用量。
- 当前周期的预估成本。
- AI 编码订阅的剩余额度。

## 开始使用

1. 在 Atmos 中打开 **AI Usage**。
2. 长时间 Agent 运行后查看最近会话。
3. 将预估结果与服务商账单页面对比。

## 使用建议

- 把应用内数字当作规划估算；服务商发票才是最终依据。
- 在启动大型重构或批量 Review 前先检查用量。
