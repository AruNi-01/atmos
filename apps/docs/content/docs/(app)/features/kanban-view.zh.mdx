---
title: Kanban 视图
description: 在任务很多时，用看板管理工作空间状态、优先级和标签。
---

Kanban 视图会把你的工作空间变成一块看板：用列表示状态，用标签表示主题，用优先级决定下一步该做什么。

## 为什么重要

- 一眼看到多个 Agent 或人工任务。
- 快速发现被阻塞或停滞的工作。
- 不用额外项目管理工具，也能协调并行工作空间。
- 拖动卡片，而不是在列表里查找。

## 可以跟踪什么

- 工作空间状态，例如 active、blocked、done 和自定义列。
- 多个任务竞争时的优先级。
- 用来表示领域、仓库或 Agent 类型的标签。

## 开始使用

1. 从 Atmos 打开 **Kanban**。
2. 把每个工作空间放到合适的列中。
3. 需要时添加标签或优先级。
4. 打开卡片即可跳转到对应工作空间。

## 使用建议

- 让看板回答一个问题：“我接下来该做什么？”
- 不要过度打标签；轻量的列比完美分类更有用。
