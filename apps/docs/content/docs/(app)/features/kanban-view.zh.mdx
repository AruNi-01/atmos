title: Kanban 视图
description: 在看板上管理工作区状态、优先级与标签，统筹多任务。
---

Kanban 视图将工作区呈现为看板——状态列、主题标签，以及在多任务并行时决定「下一步做什么」的优先级。

## 为什么重要

- 一眼看清多个 Agent 或人工任务。
- 快速发现阻塞或过久未动的项。
- 无需单独 PM 工具即可协调并行工作区。
- 拖拽卡片代替在列表中翻找。

## 可跟踪内容

- 工作区状态（进行中、阻塞、完成及自定义列）。
- 多任务竞争时的优先级。
- 按领域、仓库或 Agent 类型的标签。

## 快速开始

1. 从 Atmos 打开 **Kanban**。
2. 将各工作区放入合适列。
3. 按需添加标签或优先级。
4. 点击卡片进入对应工作区。

## 提示

- 看板回答一个问题：「接下来做哪件？」
- 不要过度打标签——简单列比完美分类更实用。
