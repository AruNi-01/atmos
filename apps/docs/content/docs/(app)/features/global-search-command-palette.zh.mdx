全局搜索和命令面板是你在 Atmos 中快速移动的入口。跳转到工作空间、打开 Canvas 或 Kanban，或运行常用操作，不用在菜单里一层层点击。

## 为什么重要

- 深度工作时保持双手在键盘上。
- 快速打开不常用但临时需要的功能。
- 按项目名或任务名找到工作空间。
- 管理大量并行任务时减少摩擦。

## 可以用来做什么

- 查找工作空间或项目。
- 打开 Canvas、Kanban、用量或设置。
- 运行内置命令和快捷操作。
- 在大量 Agent 任务中切换上下文。

## 开始使用

1. 打开命令面板（见[键盘快捷键](/zh/reference/keyboard-shortcuts)）。
2. 输入你想找内容中的几个词。
3. 选择结果并按 Enter。

## 使用建议

- 用自然词搜索：项目名、功能名或动作词。
- 当你同时运行很多任务时，和 [多工作空间开发](/zh/features/multi-workspace) 搭配使用。
