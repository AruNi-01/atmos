全局搜索与命令面板是穿越 Atmos 的快捷键。无需点菜单即可跳到工作区、打开 Canvas 或 Kanban、执行常用操作。

## 为什么重要

- 深度工作时双手留在键盘上。
- 快速打开不常用但需要的功能。
- 按项目或任务名查找工作区。
- 管理大量并行任务时减少摩擦。

## 适用场景

- 查找工作区或项目。
- 打开 Canvas、Kanban、用量或设置。
- 运行内置命令与快捷操作。
- Agent 密集会话中切换上下文。

## 快速开始

1. 打开命令面板（见 [键盘快捷键](/zh/reference/keyboard-shortcuts)）。
2. 输入目标的几個关键词。
3. 选择结果并回车。

## 提示

- 用自然词搜索：项目名、功能名或动词。
- 与 [多工作区开发](/zh/features/multi-workspace) 配合管理并行任务。
