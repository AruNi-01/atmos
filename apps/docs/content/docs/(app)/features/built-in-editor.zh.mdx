内置编辑器用于在 Atmos 内快速查看与小改——无需离开工作区、终端或 Agent 流程。打开文件、改几行，完成后回到终端或聊天即可。

## 为什么重要

- 在终端输出与 Agent 回复旁阅读代码。
- 无需打开其他 IDE 即可修正小问题。
- 任务导向的工作保持在一个界面内。
- 大型重构仍可随时回到主编辑器。

## 能做什么

- 浏览工作区文件树。
- 语法高亮预览源码。
- 行内编辑聚焦改动。
- 从 diff 或 Agent 建议跳转到文件。

## 快速开始

1. 打开工作区。
2. 从项目树或 Git diff 选择文件。
3. 在内置编辑器中阅读或修改。
4. 在终端运行检查后再提交。

## 提示

- 适合审查级小改；多文件大改用主 IDE。
- 与 [集成 Git 工作流](/zh/features/integrated-git-workflow) 配合提交前检查。
