---
title: 内置轻量编辑器
description: 预览文件、进行行内编辑，并随时切回手动编码。
---

内置编辑器适合在 Atmos 里快速查看和做小改动，不用离开工作空间、终端或 Agent 流程。打开文件，调整几行，然后回到终端或聊天继续工作。

## 为什么重要

- 在终端输出和 Agent 回复旁边阅读代码。
- 不打开另一个 IDE，也能修 typo 或小逻辑问题。
- 为聚焦任务留在同一个工作界面。
- 大型重构时仍然可以随时切回你的主编辑器。

## 你可以做什么

- 浏览工作空间文件树。
- 预览带语法高亮的源文件。
- 针对小范围改动直接编辑。
- 从 diff 或 Agent 建议跳到对应文件。

## 开始使用

1. 打开一个工作空间。
2. 从项目树或 Git diff 中选择文件。
3. 在内置编辑器里阅读或编辑。
4. 提交前在终端中运行检查。

## 使用建议

- 用 Atmos 做 Review 级别的小改动；多文件重写仍然使用你的主 IDE。
- 和 [集成 Git 工作流](/zh/features/integrated-git-workflow) 搭配，在提交前检查变更。
