Choose how you run Atmos:

## Desktop app

### macOS with Homebrew

```bash
brew install --cask AruNi-01/tap/atmos
```

### macOS install script

```bash
curl -fsSL https://install.atmos.land/install-desktop.sh | bash
```

The install script is for macOS. For Windows and Linux, download the latest installer from [GitHub Releases](https://github.com/AruNi-01/atmos/releases).

## Open in browser

Use the hosted web app when you want to open Atmos directly in a browser.

## First workspace

1. Launch Desktop or open Atmos in your browser.
2. Create or open a workspace and connect your project.
3. Open a terminal and start working with your Code Agent.

## From source (contributors)

```bash
bun install && cargo fetch
just dev-api   # terminal 1
just dev-web   # terminal 2
```
